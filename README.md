# AlignmentProtocol
A proposed form of negotiation between policy frameworks for neutral statement composition, intended as a framework to help resolve conflicts. Although this was developed using language models for feedback, research support, and as a drafting aid, the framework itself is based on novel contributions by the author developed in piecemeal over several years preceding this publication. This AlignmentProtocol project doesn’t currently have an economic basis, intended as a high level Proposal that I hope could have potential to contribute to peace in some way.

Please see document Proposal.md for details.
